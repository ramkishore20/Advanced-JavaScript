# User Management System & JSON Handling

##  About This Project

This project is part of my JavaScript learning journey. In these tasks, I practiced working with classes, objects, arrays, loops, and JSON. The goal was to understand how to create user objects, store them in an array, display their information, and work with JSON data by converting it to JavaScript objects and back.

Instead of only learning the theory, I built small programs to understand how these concepts are used in real applications.

---

##  Task 1: User Management System

In this task, I created a `User` class with the following properties:

* Username
* Mobile Number
* Age

The class also contains two methods:

* **getUserDetails()** – Returns the user's information in a formatted way.
* **isAdult()** – Checks whether the user is 18 years or older.

After creating multiple user objects, I stored them inside an array and used a loop to display each user's details along with their adult status.

---

##  Task 2: JSON Handling

In this task, I learned how JavaScript works with JSON.

The steps included:

* Creating user data as a JSON string.
* Converting the JSON string into JavaScript objects using `JSON.parse()`.
* Displaying the user information using a loop.
* Converting the JavaScript objects back into JSON using `JSON.stringify()`.

This helped me understand how data is exchanged between applications and servers.
